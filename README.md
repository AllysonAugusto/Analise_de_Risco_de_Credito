# Análise de Risco de Crédito com Machine Learning

Este projeto tem como objetivo realizar a **análise de risco de crédito**, utilizando algoritmos de **Machine Learning** para prever se um cliente irá ou não se tornar inadimplente, com base em dados financeiros e pessoais.

---

## Objetivo

- Analisar dados de clientes
- Prever o risco de inadimplência
- Aplicar algoritmos de classificação
- Avaliar o desempenho dos modelos

---

## Conceitos Abordados

- Machine Learning supervisionado  
- Classificação  
- Pré-processamento de dados  
- Treinamento e teste de modelos  
- Avaliação de modelos  

---

## Algoritmos Utilizados

- Support Vector Machine (SVM)

---

## Tecnologias e Bibliotecas

- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- Google Colab / Jupyter Notebook  

---

## Avaliação do Modelo

O desempenho do modelo é avaliado por meio de métricas como:

- Accuracy  
- Precision  
- Recall  
- F1-score  

Essas métricas permitem analisar a qualidade das previsões realizadas pelo modelo.
